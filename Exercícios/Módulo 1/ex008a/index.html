Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em destaque, usando o <strong> (forte).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag semântica <em> (emphasis).

Também é possível criar um texto marcado/highlighted usando a tag <mark> (marked).

Texto pequeno

Estamos criando um texto pequeno nesse parágrafo com a tag <small>.

Texto deletado

Para fazer um texto deletado/riscado usa-se a tag <del> (deleted).

Texto inserido

É possível marcar um texto como inserido para indicar que foi adicionado depois, usando a tag <ins> (inserted).

Subscrito e sobrescrito

Podemos escrever texto subscrito (H2O) e texto sobrescrito (x2+3 = 0) com as tags <sub> (subscripted) e <sup> (superscripted).

PARA ENVELOPAR (adicionar tags antes e depois de um texto já escrito) BASTA SELECIONAR O TEXTO, APERTAR ctrl+shift+p, SELECIONAR "wrap with abbreviation" E DIGITAR A TAG DESEJADA
