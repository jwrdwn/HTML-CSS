Links externos

Sobrepondo a guia

Com a tag <a> e o parâmetro href="" é possível criar um link para um site externo, que abrirá na aba ativa.

Acesse meu GitHub!

Em guia nova

Para abrir links em nova guia, deve-se adicionar os parâmetros target="_blank" rel="external" após o link contido na tag <a>.

Acesse meu canal do YouTube!

Links externos em outro idioma

Com o parâmetro hreflang="" é possível indicar o idioma da página linkada, para facilitar a tradução automática do usuário se for necessário.

Acesse a W3Schools!

Links internos

Para linkar páginas do próprio site, basta dar de referência o nome do arquivo.

É interessante também adicionar o parâmetro rel na tag <a>. Para a página seguinte, o comando é rel="next"; para a anterior é rel="prev".

Essa é a primeira página do site. Se desejar acessar a segunda página, clique aqui.

Parâmetros

É possível incluir outros parâmetros rel para os links:

author: indica que é um link para o site do autor do artigo atual

nofollow: indica que é um link para um site não endossado, como um link pago
