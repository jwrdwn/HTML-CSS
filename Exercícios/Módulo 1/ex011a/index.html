Imagens dinâmicas

Tente abrir esse site em dispositivos diferentes, ou aumente e diminua o tamanho do navegador!

Para inserir imagens dinâmicas, deve-se adicionar a tag <picture>, e dentro dela a imagem, usando a tag <img>. Acima da tag <img>, deve-se adicionar um source media type contendo a imagem que deve ser mostrada em tamanhos menores.

Mudar o min-width para max-width e considerar 50px acima do tamanho da imagem. A cada mudança um novo source media type deve ser adicionado.

Os source media types devem ser ordenados da imagem menor para a maior.

Áudios

Para reproduzir áudios em HTML5 existe a forma mais simples e a mais completa

Simples:

        
    <audio src="audio.mp3" controls autoplay></audio>

O parâmetro controls exibe um player para controlar o áudio e o autoplay dá play automaticamente

É possível que o autoplay não funcione em alguns navegadores.

Completo:

Para colocar um áudio da forma mais compatível, deve-se inserir a tag <audio> e remover o parâmetro src.

Dentro da tag deve ser inserida a tag <source> e completar com a origem e o tipo do áudio.

É possível colocar múltiplos sources com áudios em formatos diferentes, caso o navegador não carregue uma das mídias anteriores.

Na tag de audio, existem parâmetros a serem adicionados:

  • preload="" - indica o tipo de pré carregamento
    • auto: carrega todo o conteúdo da página, não é muito recomendado pois pode ser lento.
    • metadata: carrega informações básicas.
    • none: não carrega nada.
  • autoplay
  • controls
  • loop

Exemplo

        
    <audio preload="metadata" controls autoplay>
        <source src="audio.mp3" type="audio/mpeg">
        <source src="audio.wav" type="audio/wav">
        <source src="audio.ogg" type="audio/ogg">
        <p>Seu navegador não reproduz áudio :(</p>   
    </audio>
