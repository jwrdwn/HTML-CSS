Outras formatações

Código-fonte

O comando printf("Olá, Mundo!"); é escrito em C. Fazemos isso com a tag <code>, que gera letras monoespaçadas.

Se o desejado for representar um código completo, incluindo identação e tudo que foi originalmente escrito no HTML, usa-se a tag <code> dentro de uma tag <pre>, que mantém a formatação.

            
    int main(){
        int idade;

        printf("Digite sua idade: ");
            scanf("%d", &idade);

        return 0;
    }

Citações

Simples

Para dar significado de citação à um texto, usa-se a tag <q>: You miss 100% of the shots you dont take. -Wayne Gretzky -Michael Scott

A tag <q> apenas faz uma citação diretamente na linha do texto.

Completas

Para citações em blocos e mais completas, usa-se a tag <blockquote>

You miss 100% of the shots you dont take. -Wayne Gretzky -Michael Scott

É possível também referenciar implicitamente a citação usando <blockquote cite="link"> na tag de abertura

Abreviações / Significados

É possível mostrar um texto ou significado de uma sigla ou abreviação ao passar o mouse por cima, usando a tag <abbr>
