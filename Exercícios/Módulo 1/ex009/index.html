Usando listas

Listas ordenadas

São listas em que a ordem importa. É feita com a abertura da tag <ol> (ordered list) e cada item da lista deve estar dentro da tag <li> (list item).

  1. Acordar
  2. Tomar café
  3. Limpar a casa

Para mudar o formato da lista, como por exemplo números romanos, letras, entre outros, acrescenta-se o parâmetro type na abertura da ol.

Usa-se "a" ou "A" para listas a, b, c / A, B, C; "1" para listas 1, 2, 3; "i" ou "I" para listas i, ii, iii / I, II, III

Para começar a lista em um item que não seja o primeiro, acrescenta-se o parâmetro start="" na tag <ol>

  2. Tomar café
  3. Limpar a casa

Listas não ordenadas

É usado em listas onde a ordem não importa. A tag é <ul> (unordered list)

  • Arroz
  • Feijão
  • Macarrão
  • Ovo
  • Café

O ícone padrão de listas não ordenadas é o bullet, declarado como disc. Para modificar o ícone, deve-se acrescentar o parâmetro type="" na tag <ul>.

disc: círculo preto preenchido (bullet)
circle: círculo preto vazado
square: quadrado preenchido

Listas de definição

São listas que mostram a definição de um termo É feita com a abertura da tag <dl> (definition list) e cada item da lista deve estar dentro da tag <dt> (definition term) e deve também conter uma tag <dd> (definition description).
